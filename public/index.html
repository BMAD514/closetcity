Closet.city API

This project hosts API endpoints for the Closet.city app on Cloudflare Pages Functions.

Endpoints

  • GET /api/garments
  • GET /api/garments/<id>
  • GET /api/r2/<key>
  • POST /api/upload
  • POST /api/model
  • POST /api/tryon
  • POST /api/pose

If you reached this page from the UI, set NEXT_PUBLIC_API_BASE_URL in the UI project to this domain so the frontend calls these endpoints.
